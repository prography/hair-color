## 1. 딥러닝 이슈
### 1.1 hair image 데이터 모으는 방법

+ 젤 중요한 부분. 충분한 데이터를 모아야한다. 

+ 논문에서는 돈 주고 사거나 클라우드 펀딩을 받음 

+ 데이터를 모으지 못한다면 주제를 변경할 수도 있다.



### 1.2 논문에 쓰인 ρ = 0.95 를 그대로 쓸 것인가
---------------
 + MobileNet에서 나오는 ρ는 해상도를 줄여 속도를 높이는 값이다.

 + ρ = 0.95 로 한다면 속도가 줄어들지 알아보자
